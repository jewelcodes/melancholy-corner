# [Dead Project]

**[Melancholy Corner](https://melancholy.xyz)** is an online radio station that broadcasts vaporwave and lo-fi music. It uses responsive CSS and JavaScript to mimic the Windows 95/98 look and feel as well, to take you back to the past.
